A. UPLOAD FILE PHP
──────────────────────────────────────────────────────────────────────
if (isset($_FILES['gambar']) && $_FILES['gambar']['error'] == 0) {
    $allowed = ['jpg', 'jpeg', 'png', 'gif'];
    $filename = $_FILES['gambar']['name'];
    $filetype = pathinfo($filename, PATHINFO_EXTENSION);
    
    if (in_array(strtolower($filetype), $allowed)) {
        $newname = time() . '_' . $filename;
        $destination = '../assets/img/menu/' . $newname;
        
        if (move_uploaded_file($_FILES['gambar']['tmp_name'], $destination)) {
            // Update database
            $gambarPath = 'assets/img/menu/' . $newname;
            $stmt = $conn->prepare("UPDATE menu SET gambar = ? WHERE id = ?");
            $stmt->bind_param("si", $gambarPath, $id);
            $stmt->execute();
            
            echo "Upload berhasil!";
        }
    }
}

═══════════════════════════════════════════════════════════════════════════
🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

MASALAH 1: Upload gagal
──────────────────────────────────────────────────────────────────────
SOLUSI:
- Cek permission folder: chmod 755 assets/img/
- Cek php.ini:
  upload_max_filesize = 10M
  post_max_size = 10M
- Restart Apache

MASALAH 2: Icon tidak muncul
──────────────────────────────────────────────────────────────────────
SOLUSI:
- Cek koneksi internet (Font Awesome dari CDN)
- Cek kode icon benar: "fas fa-home" bukan "fa-home"
- Refresh browser (Ctrl + F5)

MASALAH 3: Foto tidak tampil
──────────────────────────────────────────────────────────────────────
SOLUSI:
- Cek path foto di database benar
- Cek file foto ada di folder
- Cek permission file: chmod 644
